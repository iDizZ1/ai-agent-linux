## Категория: Работа с файлами

- `touch file.txt` - создать пустой файл
- `cat file.txt` - показать содержимое файла
- `cat > file.txt` - записать текст в файл (Ctrl+D для завершения)
- `echo "text" > file.txt` - записать текст в файл перезаписывая содержимое
- `echo "text" >> file.txt` - добавить текст в конец файла
- `nano file.txt` - открыть файл в редакторе nano
- `vim file.txt` - открыть файл в редакторе vim
- `head -n 10 file.txt` - показать первые 10 строк файла
- `tail -n 10 file.txt` - показать последние 10 строк файла
- `wc -l file.txt` - подсчитать количество строк в файле
- `wc -c file.txt` - подсчитать количество байт в файле
- `wc file.txt` - показать строки, слова, байты

## Категория: Работа с директориями

- `mkdir folder_name` - создать новую папку
- `mkdir -p path/to/folder` - создать папку со всеми родительскими директориями
- `cd folder_name` - перейти в папку
- `cd ..` - перейти в родительскую папку
- `cd ~` - перейти в домашнюю папку
- `cd /` - перейти в корневую папку
- `pwd` - показать текущий путь
- `ls` - показать список файлов и папок
- `ls -la` - показать скрытые файлы и подробную информацию
- `ls -lh` - показать размер файлов в человеческом формате
- `ls -R` - рекурсивный список всех файлов и папок
- `ls -t` - отсортировать по времени изменения
- `ls -S` - отсортировать по размеру
- `rmdir folder_name` - удалить пустую папку
- `rm -r folder_name` - удалить папку со всем содержимым
- `tree folder_name` - показать структуру папок в виде дерева
- `find . -type d` - найти все папки в текущей директории и подпапках

## Категория: Поиск и фильтрация

- `find . -name "*.txt"` - найти все txt файлы в текущей папке
- `find / -name "file.txt"` - найти файл по всей системе
- `find . -type f -name "*.py"` - найти все python файлы
- `find . -size +100M` - найти файлы больше 100 МБ
- `find . -mtime -1` - найти файлы измененные за последний день
- `find . -perm 777` - найти файлы с определенными правами доступа
- `grep "pattern" file.txt` - найти строки содержащие паттерн
- `grep -r "pattern" .` - рекурсивный поиск в файлах
- `grep -i "pattern" file.txt` - поиск без учета регистра
- `grep -v "pattern" file.txt` - показать строки БЕЗ паттерна
- `grep -c "pattern" file.txt` - подсчитать количество совпадений
- `grep -n "pattern" file.txt` - показать номера строк с совпадениями
- `locate filename` - быстрый поиск файла в индексе
- `which command_name` - найти путь к команде

## Категория: Текстовая обработка

- `sed 's/old/new/g' file.txt` - заменить old на new (все вхождения)
- `sed -i 's/old/new/g' file.txt` - заменить в файле (изменить файл)
- `sed '5d' file.txt` - удалить 5-ую строку
- `sed '1,5d' file.txt` - удалить строки с 1 по 5
- `sort file.txt` - отсортировать строки в файле
- `sort -r file.txt` - отсортировать в обратном порядке
- `sort -n file.txt` - отсортировать как числа
- `sort -u file.txt` - отсортировать и удалить дубликаты
- `uniq file.txt` - удалить повторяющиеся строки
- `cut -d',' -f1 file.csv` - вырезать первый столбец из csv
- `awk '{print $1}' file.txt` - вывести первое поле каждой строки
- `awk -F',' '{print $2}' file.csv` - установить разделитель полей на запятую

## Категория: Работа с процессами

- `ps aux` - показать все запущенные процессы
- `ps aux | grep "python"` - найти процесс python
- `top` - интерактивный монитор процессов
- `htop` - улучшенный монитор процессов (если установлен)
- `kill 1234` - завершить процесс с id 1234
- `killall python` - завершить все процессы с именем python
- `kill -9 1234` - принудительно завершить процесс
- `pkill -f "pattern"` - завершить процессы по паттерну имени
- `bg` - запустить процесс в фоне
- `fg` - вернуть процесс на передний план
- `jobs` - показать список фоновых процессов
- `nohup command &` - запустить команду в фоне с игнорированием hangup
- `&` - запустить команду в фоне

## Категория: Информация о системе

- `uname -a` - показать информацию о системе
- `uname -r` - показать версию ядра
- `lsb_release -a` - показать информацию о Linux дистрибутиве
- `hostnamectl` - показать имя хоста и информацию о системе
- `whoami` - показать текущего пользователя
- `id` - показать информацию о пользователе и группах
- `groups` - показать группы текущего пользователя
- `w` - показать кто залогинен в системе
- `who` - показать список пользователей в системе
- `df -h` - показать использование дискового пространства
- `df -i` - показать использование inode
- `du -sh *` - показать размер каждой папки в текущей директории
- `du -h --max-depth=1` - показать размер папок максимум на 1 уровень
- `free -h` - показать информацию о памяти
- `top -b -n 1` - получить снимок процессов (не интерактивный)
- `vmstat 1 5` - показать статистику памяти 5 раз с интервалом 1 сек

## Категория: Права доступа и пользователи

- `chmod 755 file.txt` - установить права 755
- `chmod u+x file.txt` - добавить право на исполнение для владельца
- `chmod -R 755 folder/` - рекурсивно изменить права
- `chown user:group file.txt` - изменить владельца файла
- `chown -R user:group folder/` - рекурсивно изменить владельца
- `sudo command` - запустить команду с правами администратора
- `sudo su` - переключиться на пользователя root
- `su username` - переключиться на другого пользователя
- `whoami` - показать текущего пользователя

## Категория: Работа с архивами

- `tar -cvf archive.tar folder/` - создать tar архив
- `tar -xvf archive.tar` - распаковать tar архив
- `tar -czf archive.tar.gz folder/` - создать сжатый tar архив
- `tar -xzf archive.tar.gz` - распаковать сжатый tar архив
- `tar -cjf archive.tar.bz2 folder/` - создать tar архив с bzip2 сжатием
- `tar -xjf archive.tar.bz2` - распаковать tar.bz2 архив
- `zip -r archive.zip folder/` - создать zip архив рекурсивно
- `unzip archive.zip` - распаковать zip архив
- `gzip file.txt` - сжать файл gzip (создаст file.txt.gz)
- `gunzip file.txt.gz` - распаковать gzip файл

## Категория: Сетевые команды

- `ping google.com` - проверить подключение к хосту
- `ping -c 5 google.com` - отправить 5 ping пакетов
- `curl https://example.com` - получить содержимое веб-страницы
- `curl -I https://example.com` - получить только headers
- `wget https://example.com/file` - скачать файл
- `ifconfig` - показать информацию о сетевых интерфейсах
- `ip addr` - показать IP адреса (новый способ)
- `netstat -an` - показать все открытые порты
- `ss -an` - показать сокеты (альтернатива netstat)
- `nslookup google.com` - показать IP адрес хоста
- `dig google.com` - детальная информация о DNS
- `traceroute google.com` - показать маршрут до хоста
- `ssh user@host` - подключиться к удаленному хосту
- `scp file.txt user@host:/path/` - скопировать файл на удаленный хост
- `scp user@host:/path/file.txt .` - скопировать файл с удаленного хоста

- `python -m pytest` - запустить тесты pytest
- `python -m pytest -v` - запустить тесты с выводом
